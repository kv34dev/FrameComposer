# FrameComposer — Camera Framing & Composition App

**FrameComposer** is an iOS application built with **SwiftUI**, designed to help users capture perfectly composed photos using guided overlays and smart camera tools.  
It combines a clean interface with real-time camera preview and framing templates to elevate mobile photography.

## ✨ Features

- **📸 Live Camera Preview** — real-time camera feed with smooth performance  
- **📐 Grid Overlay** — improve composition with visual guides  
- **🧩 Templates** — choose framing presets for different shot styles  
- **🎯 Precision Framing** — align subjects perfectly in your shots  
- **⚡ Lightweight & Fast** — optimized for performance  
- **💡 Minimal UI** — distraction-free shooting experience  

## 🧩 Tech Stack

- **Language:** Swift  
- **Framework:** SwiftUI  
- **Camera Handling:** AVFoundation  
- **Architecture:** MVVM-like structure  
- **Platform:** iOS

## 🚀 Installation

1. Clone this repository:
```
git clone https://github.com/kv34dev/FrameComposer
```
2. Open the project in Xcode:
```
cd FrameComposer
open FrameComposer.xcodeproj
```
3. Build and run the app on a simulator or physical device (camera requires a real device).
